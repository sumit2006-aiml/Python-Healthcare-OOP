## 📌 Suite Overview
The **Healthcare Software Suite** is a collection of modular Python applications designed to solve real-world challenges in the pharmacy and clinical research sectors. Each module is built with a focus on **Data Integrity**, **Automated Auditing**, and **Scalable Architecture**.



---

## 📂 Project Modules

### 1. [Drug Inventory Management System (PIMS)](./Drug_Inventory_System)
A backend controller for high-volume pharmacy environments.
* **Core Logic:** Expiry date auditing, stock threshold alerts, and real-time financial valuation.
* **Key Features:** Automated QC checks and batch-wise inventory tracking.
* **Status:** 🟢 Version 1.0 (Stable)

### 2. [Clinical Trial & Patient Management](./Clinical_Trial_System)
A data-centric system for managing participant enrollment and medical history.
* **Core Logic:** Patient data encapsulation and trial eligibility filtering.
* **Key Features:** Secure participant tracking and automated status reporting.
* **Status:** 🟡 Version 0.8 (Core Logic Complete)

## 🛠️ Technical Standards
* **Design Pattern:** Object-Oriented Programming (Classes, Methods, Encapsulation).
* **Execution Safety:** All modules utilize `if __name__ == "__main__":` blocks to ensure clean imports for future GUI/Web integration.
* **Organization:** Modular folder structure aligned with Open Source best practices (GSoC 2027 standards).
